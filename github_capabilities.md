# TruContext Technical Capabilities from GitHub Repository

## Core Platform Architecture

### Dual Database Persistence
- **Neo4j Primary Storage**: Graph database for real-time network relationships and threat analysis
- **PostgreSQL Backup**: Relational database for archival, analytics, and persistent storage
- **Automatic Synchronization**: Both databases updated simultaneously for data consistency
- **Zero Data Loss**: Enterprise-grade reliability with dual database architecture

### Advanced Graph Analytics
- **Real-time Network Analysis**: Live data synchronization and threat detection
- **Interactive Network Graphs**: Drag, zoom, and explore network relationships
- **Multiple Layout Algorithms**: Cola, ELK, Klay, and custom layouts for optimal visualization
- **Dynamic Styling**: Node and edge styling based on properties and threat levels

### Geospatial Capabilities
- **Geographic Map Integration**: Geospatial threat visualization with persistent coordinate assignment
- **Coordinate Assignment Wizard**: Automated geographic coordinate assignment with dual database persistence
- **Continuous Integration**: Add geographic data to existing datasets without recreation
- **Real-time Updates**: Map refreshes immediately after coordinate assignment
- **Persistent Visualization**: Geographic data maintained across sessions and deployments

### AI-Powered Features
- **AI-Powered Icon Generation**: Dual API system with Recraft.ai primary and Gemini fallback
- **Dynamic Icon Management**: PostgreSQL-powered icon mapping system with preview and approval workflow
- **Cloudinary Integration**: Cloud-based icon storage and management
- **AI Agents**: Agentic AI capabilities for automated threat analysis
- **AI Dashboards**: AI-driven dashboards for executive and SOC teams

### Threat Analysis Capabilities
- **Threat Path Analysis**: Identify and analyze attack vectors and attack chains
- **MITRE ATT&CK Integration**: Map threats to tactics, techniques, and procedures
- **Real-time Threat Detection**: Continuous monitoring and alerting
- **Predictive Analytics**: AI-powered forecasting of potential threats

### Dashboard Views
- **Executive Dashboard**: High-level overview for leadership
- **SOC Executive Dashboard**: Security Operations Center view
- **Threat Analysis Dashboard**: Detailed threat investigation and analysis
- **Geographic Map View**: Geospatial threat visualization
- **AI Agents View**: Agentic AI workflow automation
